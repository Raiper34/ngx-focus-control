Ngx Focus Control

Focus control directive

Focus control directive allows you to control next and previous focused element of current element. Example shows how to revert order of focus using this directive. Focus element and try to navigate with tab and tab + shift.

Code

{{focusControlCode}}

Focus selector directive

Focus selector directive allows you to focus only element by given selector. Input 5 do not have this directive, so it is skipped from focus flow, but it is still accessible. Focus element and try to navigate with tab and tab + shift.
