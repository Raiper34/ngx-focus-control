Ngx Focus Control

Focus control directive

This directive allows the user to manually define the next and/or previous focus target. Example shows how to revert direction of focus. Focus the first element and navigate using the keyboard (Tab and Shift + Tab).

{{focusControlCode}}

Focus selector directive

This directive allows the user to manually define the next and previous focus target by the given class. It can be configured to behave in the cycle (if it is the last element with the given class, the next focus target is the first element with the given class). Example shows how to skip element between elements with this class (element is still accessible by mouse). Focus the first element and navigate using the keyboard (Tab and Shift + Tab).

Code

{{focusSelectorCode}}

Focus parent directive

This directive behaves very similarly to Focus selector directive, only the user can access focusable elements between elements with this directive by pressing Enter. Example shows how to focus div element and its content (content is focusable by click or enter). Focus the first div and navigate using the keyboard (Tab and Shift + Tab and Enter).
